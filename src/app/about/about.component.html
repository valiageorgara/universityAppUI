Instructions for the Applicant

Instructions to create a New Application for a Postgraduate Program

  • Click Postgraduate Programs to see available Postgraduate Programs.

  • Select one to submit an Application for a Postgraduate Program.

  • In the Form Application for a Postgraduate Program fill in your details.

  • In the Preview Application section you can see an Overview of your Application for a Postgraduate Program.

  • In the Preview Application section click Submit Application to submit your Application for a Postgraduate Program.

General Instructions of Applications for a Postgraduate Program

  • Click My Applications to see a list of your Applications for a Postgraduate Programs and their status.

  • From the Actions Column:

    • Click on to Edit your Application for a Postgraduate Program.
    • Click on to View your Application for a Postgraduate Program.
    • Click on to see an Overview of your Application for a Postgraduate Program and Submit it.
